================================================================================
  DEPLOYMENT STEPS
================================================================================

1. Create Upstash Database (5 minutes)
   - Visit https://console.upstash.com
   - Create Redis database (us-east-1 recommended)
   - Copy REST URL and TOKEN

2. Set Environment Variables (Vercel)
   UPSTASH_REDIS_REST_URL=https://your-endpoint.upstash.io
   UPSTASH_REDIS_REST_TOKEN=your-token-here

3. Verify Tests (< 1 minute)
   pytest tests/unit/ tests/integration/ tests/e2e/ -v

4. Deploy
   git push origin working-deployment

5. Monitor (first 30 minutes)
   - Check logs for "✅ Connected to Upstash Redis"
   - Verify rate limiting across replicas
   - Monitor cache hit ratios
   - Check Upstash dashboard for activity
