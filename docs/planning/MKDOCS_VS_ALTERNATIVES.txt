🔑 KEY DIFFERENCE: AUTO API DOCS
================================================================================

THE PROBLEM:
  Atoms MCP has 5 tools with complex parameters
  Each tool needs: signature, parameters, types, returns, examples, errors
  Manual documentation: 2-3 hours per tool = 10-15 hours total

WITH SPHINX AUTODOC:
  ✅ Write docstring once (30 min)
  ✅ Sphinx auto-generates complete API docs (0 min)
  ✅ Docs always in sync with code
  ✅ Total: 2.5 hours for all 5 tools

WITHOUT SPHINX (Docusaurus/Fumadocs):
  ❌ Write docstring (30 min)
  ❌ Manually write API docs (120 min per tool)
  ❌ Manually write examples (30 min per tool)
  ❌ Manually maintain (30 min per update)
  ❌ Total: 15 hours for all 5 tools

TIME SAVINGS: 12.5 hours per project = 100+ hours over 3 years

💡 REAL EXAMPLE: entity_operation TOOL
================================================================================

SOURCE CODE (Python):
  async def entity_operation(
      operation: str,
      entity_type: str,
      entity_id: Optional[str] = None,
      properties: Optional[Dict[str, Any]] = None,
      filters: Optional[Dict[str, Any]] = None,
      limit: int = 100,
      offset: int = 0,
  ) -> Dict[str, Any]:
      """Perform CRUD operations on entities.
      
      Args:
          operation: The operation to perform
          entity_type: The type of entity
          entity_id: The ID of the entity
          properties: Entity properties
          filters: Filters for list/search
          limit: Maximum results (default: 100)
          offset: Pagination offset (default: 0)
          
      Returns:
          Operation result with entity data
      """

WITH SPHINX AUTODOC:
  ✅ Run: sphinx-apidoc -o docs/api ../server.py ../tools/
  ✅ Generates complete API documentation automatically
  ✅ Includes: signature, parameters, types, returns, examples
  ✅ Time: 30 seconds

WITHOUT SPHINX:
  ❌ Manually write API documentation
  ❌ Manually create parameter table
  ❌ Manually write examples
  ❌ Manually maintain when code changes
  ❌ Time: 2-3 hours
